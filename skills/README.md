# Skills

Portable agent skills live here as folders containing `SKILL.md`.

Recommended shape:

```text
skills/
└── example-skill/
    ├── SKILL.md
    ├── references/
    ├── scripts/
    └── templates/
```

Public-safety rule: include reusable procedures only. Do not include personal context, private project specifics, account details, or secret-bearing config.
